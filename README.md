# Heart Disease Prediction with Model Tuning

## Overview
Binary classification project to predict heart disease presence, 
with focus on model evaluation techniques: Cross-Validation and 
Hyperparameter Tuning (GridSearchCV).

## Dataset
Heart Failure Prediction Dataset (Kaggle - fedesoriano)
918 rows, 12 features, binary target (HeartDisease: 0/1)

## What This Project Covers
- Exploratory Data Analysis
- Data preprocessing & encoding
- Baseline model (Logistic Regression)
- Cross-Validation (K-Fold)
- Hyperparameter Tuning (GridSearchCV)
- Model comparison: before vs after tuning
- Overfitting/Underfitting analysis

## Tools
Python, Pandas, scikit-learn, Matplotlib/Seaborn
